# Step 1. 데이터셋 구성

| 항목 | 값 |
| --- | ---: |
| 원본 행 수 | 500 |
| 전처리 후 행 수 | 485 |
| 전처리 후 컬럼 수 | 26 |
| 제외 행 수 | 15 |

## 입력과 출력

- 원본: `data/raw/ap_whellchair_hall_total.csv`
- 전처리 결과: `data/processed/accessibility_dataset.csv`

## 전처리 기준

- 날짜와 좌석/예매 수를 명시적 타입으로 변환했다.
- 장애인석, 총좌석, 유료입장객 수가 0 이하인 행은 보정 예매율을 안정적으로 계산할 수 없어 제외했다.
- 텍스트 결측값은 `unknown`으로 채우고 앞뒤 공백을 제거했다.
